== Installation ==

= First time installation =
1. Upload the FOLDER 'wp-easy-slider' to the /wp-content/plugins/
2. Activate the plugin 'WP Easy Slider' through the 'Plugins' menu in admin
3. Go to 'Manage / Easy Slider' for more instructions

= Upgrade from a previous version =
1. Deactivate 'WP Easy Slider' through the 'Plugins' menu in admin
2. Upload the FOLDER 'wp-easy-slider' to the /wp-content/plugins/
3. Activate the plugin 'WP Easy Slider' through the 'Plugins' menu in admin

If you have any problem just <a href="http://www.jordisalord.com/wp-easy-slider">leave a comment</a> describing the problem.
